# 📩 SMS Spam Classifier

A Machine Learning web application that classifies SMS messages as **Spam** or **Not Spam**.

## 🚀 Live Demo
👉 https://sms-spam-classifier-coi6.onrender.com

---

## 🧠 Machine Learning Model
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- NLTK for text preprocessing

---

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- Render (Deployment)
